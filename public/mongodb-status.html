Hướng dẫn kết nối MongoDB Atlas

  1. Cập nhật file .env với connection string từ MongoDB Atlas
  2. Đảm bảo đã cài đặt MongoDB driver: npm install mongodb
  3. Khởi động server: node server.js
  4. Kiểm tra kết nối tại: http://localhost:1000/api/debug/db-connection
  5. Nếu kết nối thành công, ứng dụng sẽ sử dụng MongoDB Atlas thay vì dữ liệu tạm

Connection string mẫu:

mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/database?retryWrites=true&w=majority

Xử lý lỗi thường gặp:

  • Đảm bảo đã whitelist IP trong MongoDB Atlas
  • Kiểm tra username và password trong connection string
  • Đảm bảo đã chỉ định tên database trong connection string
